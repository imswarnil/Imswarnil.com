## 5. `post.hbs` is now a dispatcher + content-type layouts

`post.hbs` reads the post's internal (`#`) tag and renders a dedicated layout from `partials/post/types/`.

| `#tag` | Layout partial | URL |
|---|---|---|
| `#course` | `course-detail` (overview + auto lesson list + CTA) | `/course/{slug}/` |
| `#lesson` | `course-lesson` (body + sticky lesson rail) | `/course/{primary_tag}/{slug}/` |
| `#video` | `video` (playlist + episode grid) | `/videos/{slug}/` |
| `#video-detail` | `video-detail` (player + more from playlist) | `/videos/{primary_tag}/{slug}/` |
| `#travel` | `travel` (full-bleed photo story) | `/travel/{slug}/` |
| `#guide` | `guide` (long-form + TOC + "Updated") | `/guides/{slug}/` |
| `#project` | `portfolio` (case study + meta) | `/projects/{slug}/` |
| `#product` | `product` (showcase + related) | `/products/{slug}/` |
| *(none)* | `standard` (original article layout) | `/{slug}/` |

**Pages (auto-applied by slug):** `page-about.hbs` (slug `about`), `page-now.hbs` (slug `now`, renders a timeline from `#timeline` posts).

**Files:** `post.hbs`, `partials/post/header.hbs`, `partials/post/types/*.hbs`, `page-about.hbs`, `page-now.hbs`, `assets/css/pages/post-types.css`, `routes.yaml` (+ synced to `content/settings/routes.yaml`).

**Convention:** a course and its lessons share the same **public primary tag** (e.g. `javascript`); same for a video playlist and its `#video-detail` videos тАФ that's how they auto-link.

**Validate**
1. Create the internal tags in Ghost Admin (e.g. `#course`, `#lesson`, `#video`, `#video-detail`, `#travel`, `#guide`, `#project`, `#product`, `#timeline`).
2. Create a test post, add one type tag (e.g. `#guide`) + publish тЖТ open it; confirm the layout matches the table above.
3. For course/lessons: give the course post tags `#course` + `javascript`; give lessons `#lesson` + `javascript`. Open the course тЖТ lessons list appears; open a lesson тЖТ breadcrumb back + lesson rail with current highlighted.
4. Create pages with slugs `about` and `now` тЖТ visit `/about/` and `/now/` (they 404 until the pages exist тАФ that's expected).
5. `npm run test` should pass.
